# Legend of Noera

The official website of *Legend of Noera*—a fantasy saga about dignity, God-given Gifts, neurodiversity, and one girl's journey between Earth and Zionara.

The static site includes a lightweight page-by-page novel reader with four browser narrator profiles.

After GitHub Pages is enabled: https://jongpogi87.github.io/Legend-Of-Noera/
